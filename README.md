# 海克斯乱斗实验室

## 项目简介

海克斯乱斗实验室是一个专注于《英雄联盟》ARAM Mayhem（海克斯大乱斗）模式的**增强符文数据库与资源站**。项目核心是回答玩家最关心的问题：**"这个增强符文具体有什么效果？搭配什么英雄最强？"**

项目围绕增强符文图鉴展开，涵盖效果数值详解、英雄推荐组合、避雷指南、已知 Bug 追踪等功能。数据来源于 arammayhem.com、blitz.gg 等公开数据平台，所有数据均经过交叉验证。

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML / CSS / JavaScript | 纯前端 SPA，数据通过 `fetch()` 加载 JSON |
| Chart.js 4.4.1 | 数据可视化图表（CDN） |
| Python 3 | 数据采集、清洗、校验脚本 |
| arammayhem.com | 增强符文效果描述、胜率/选取率数据 |
| blitz.gg | 英雄推荐组合、统计数据 |
| Data Dragon | 英雄头像等静态资源 |

## 功能模块

### 增强图鉴（核心）
完整的增强符文数据库，每个增强展示：中英文名称、稀有度徽章（棱彩/黄金/白银）、效果描述（含具体数值和 AD/AP 加成）、标签分类、胜率/选取率、英雄推荐（T0/T1/T2 排序）。支持搜索、按稀有度和状态筛选、展开详情。

### 英雄档案
172 个英雄的完整档案，展示：胜率/选取率/KDA/对局数统计、出装建议、使用技巧、增强协同分析（质变/推荐/避雷/疑似Bug 四个等级）。英雄推荐按稀有度分组、按推荐等级排序。

### 组合探索
英雄 × 增强协同矩阵，每条数据标注质变/推荐/避雷/疑似 Bug 等级，附带置信度百分比。

### 问题追踪
8 条经过验证的已知 Bug 结构化追踪，包含严重程度、涉及英雄/增强、当前状态、规避建议。

## 快速开始

### 查看网站

```bash
# 使用 Python 内置 HTTP 服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 数据校验

修改 `data/*.json` 后运行校验确保数据完整性：

```bash
python scripts/verify_data.py
python scripts/final_validation.py
```

校验内容：重复 ID、稀有度/状态枚举值合法性、必需字段完整性、胜率格式一致性。

### 数据更新流程

增强数据的更新遵循以下安全流程：

1. **抓取**：从 arammayhem.com 抓取原始数据 → 输出到 `pipeline/output/`
2. **校验**：审核候选数据，确认准确性
3. **写入**：通过脚本将审核后的数据写入 `data/augments.json`
4. **验证**：运行校验脚本确认无数据损坏
